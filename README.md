# The Marine

이 모드는 하츠오브아이언에 해병문학에 등장하는 포항 해병대의 국가를 추가한 모드이며 이름은 The Marine 입니다.

## 설치 방법

### 스팀에서 다운로드

1. 스팀 링크 (https://steamcommunity.com/sharedfiles/filedetails/?id=2913420947&searchtext=Marine)에서 모드를 다운로드 합니다.
2. 플레이세트에 모드를 추가합니다.
3. 게임을 실행합니다.

### 직접 추가하기

1. `git clone https://github.com/Murak549/MarineRedux.git` 으로 파일 사본을 저장합니다.
2. 런처 실행 후 모드를 생성합니다.
3. Hearts of Iron IV의 모드 폴더에 모드 파일을 추가합니다. 모드 폴더는 Hearts of Iron IV가 설치된 폴더 안에 있는 "mod" 폴더 내의 당신이 설정한 폴더 입니다.
4. Hearts of Iron IV를 실행하고, "The Marine" 모드를 활성화합니다.
